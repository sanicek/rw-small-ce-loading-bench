[h1]Small CE Loading Bench[/h1]

Give Combat Extended's loading bench a compact 1x1 footprint while preserving its normal ammunition recipes, bills, and behavior.

[h2]Features[/h2]

[list]
[*]Reduces Combat Extended's existing loading bench from 3x1 to 1x1
[*]Preserves the bench identity, recipes, bills, costs, and research integration
[*]Lets placement rotation move the interaction spot to any side
[*]Keeps one consistent fixed-perspective graphic with normal stuff coloring
[*]Uses a narrow XML patch with no additional runtime framework
[/list]

[h2]Requirements[/h2]

Targets RimWorld 1.6 and requires:

[list]
[*][url=https://steamcommunity.com/sharedfiles/filedetails/?id=2890901044]Combat Extended[/url]
[/list]

Use RimWorld's automatic mod sorting before starting the game.

[h2]Existing colonies and compatibility[/h2]

Existing Combat Extended loading benches become one-cell buildings while this mod is enabled. The original bench identity remains unchanged, so existing bills and recipes stay attached.

Mods that also change Combat Extended's AmmoBench footprint, graphics, draw size, or rotation behavior may conflict. Recipe additions should generally remain compatible because they continue to target the same bench.
