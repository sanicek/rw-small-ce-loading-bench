# Version 0.1.0

Status: Accepted

## Summary

Initial release of Small CE Loading Bench, reducing Combat Extended's loading
bench to a one-cell workstation while preserving its established behavior.

## Changes

- Patch CE's existing `AmmoBench` to use a 1x1 footprint.
- Use a fixed-perspective, stuff-recolored `Graphic_Single` while placement
  rotation continues to control the interaction spot.
- Preserve CE recipes, bills, costs, research, facilities, and Def identity.
- Add the canonical cube workbench with muted powder-hopper and loading-press
  fixtures, preserving stuff recoloring on the bench body.

## Update risks

This is the first release. Existing CE loading benches become one-cell buildings
when this mod is enabled; no Def, bill, or recipe identity is replaced.
